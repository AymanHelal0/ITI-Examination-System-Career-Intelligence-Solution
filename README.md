# ITI Examination System & Career Intelligence Solution

An end-to-end digital solution developed as an ITI graduation project, designed to automate the examination lifecycle and extend beyond grading to deliver data-driven career intelligence.

This system transforms educational data into strategic insights that help measure educational ROI, student performance, and real-world career outcomes.
